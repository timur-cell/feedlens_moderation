## Environment notes / deviations

- **Seed count**: the TS seed (`convex/seedAllRules.ts`) contains **81** rules, not the 79 the
  spec mentions; the TS code wins, so seeds insert 81 (one of them, `too_low_price_430k`,
  ships disabled — also TS parity).
- Docker Hub anonymous pulls are rate-limited in this environment; base images use
  `mirror.gcr.io/library/*` (byte-identical mirrors of Docker Hub official images);
  override with `BASE_REGISTRY=docker.io`.
- Sandbox egress blocks `www.jamesedition.com` / `img.jamesedition.com` and `api.openai.com`;
  live integrations against those hosts are implemented and unit-tested via WebMock but can
  only be smoke-tested outside this environment.
- No `ANTHROPIC_API_KEY` / `OPENAI_API_KEY` available here → AI integrations are
  "verified mocked only" (criterion 4 wording).
